# Git Tutorial

This repository is part of the mini-workshop given to the Ballen Research Group
about the basics of Git. The complete tutorial contains the following files:

- `Anolis.tre` and `svl.csv`, sourced from the [phytools tutorial](http://www.phytools.org/***SanJuan2016/ex/7/Anc-states-continuous.html) on ancestral state reconstruction;
- The scripts `anoles.R` and `properties.R`, also sourced from the tutorial, but slightly modified;
- A `Makefile` to automate execution and clean-up of scripts;
- A `.gitignore` to inhibit the upload of unnecessary PDF files.

Your objective is to fork this repository, create your own branch from `starting-point`, achieve a similar state to the main branch and create a pull request to this repo with your changes.

You can find the accompanying presentation on the subject [here](https://docs.google.com/presentation/d/1tmrbmdcxf-hWNw8Z2l5lwDFb5qtAHteZdvDD1wp_f1I/edit?usp=sharing).
